## Getting Started

- To get a local copy up and running follow these simple steps

### Installation

1. Clone the repository

```sh
git clone https://github.com/ZAD4YTV/Electron-Learning/
```

1. Move inside the repository

```sh
cd Electron-Learning
```

1. Start the App

- With Node:

```sh
npx run start
```

- Or with Yarn:

```sh
yarn start
```
